# kitty documentation.
Manual installation **DEPRECATED**:
```
git clone https://github.com/a2n-s/kitty ~/repos/kitty
cd ~/repos/kitty
git checkout main
make
ln -s ./kitty/launcher/kitty /usr/bin/kitty
```
One can also refer to the [GitHub repo](https://github.com/a2n-s/kitty).
