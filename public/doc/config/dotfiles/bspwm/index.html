“bspwm is a tiling window manager that represents windows as the leaves of a full binary tree.”

The Files.⌗

All files are available at ~/.config/bspwm:

  • bspwmrc
  • scheme.sh

The config.⌗

Some scripts can be found in bspwm’s config.
bspwm supports polybar very well as in the above images. For an updated version of my polybar see the repo or the doc.
For all the key-bindings of bspwm see sxhkd.

Some resources.⌗

  • the GitHub page.
  • the Arch wiki page.
  • the Arch manual page.
  • inspired by notusknot’s video on the topic and his repo and also aguslr’s and dt’s configs.

Dependencies.⌗

  • feh.
  • bspc.
  • sxhkd.
  • alacritty.

Manual installation process.⌗

  • install all the dependencies above.
  • move the content of bspwm's config to your ~/.config/bspwm directory.

in a nutshell and to be adapted:

sudo pacman -Syu feh bspwm sxhkd alacritty
git clone git@github.com:a2n-s/dotfiles.git a2n-s_dotfiles
mv a2n-s_dotfiles/.config/bspwm/* ~/.config/bspwm
